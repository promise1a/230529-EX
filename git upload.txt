1. 일단 해당 폴더에
git clone 원격지주소
입력해서 동일화시켜준다.

2. .git이 생긴 폴더에서 다시 git bash를 연다.

3. git init :: 연결시켜준다

4. git branch b1 :: 브런치를 새로 생성시킨다.(보통 기존엔 main)

5. git checkout b1 :: 브런치 b1으로 이동한다.

6. git add . :: 폴더 전체를 작업 영역으로 지정한다.

7. git commit -m "repos" :: "" 안에 변경사항을 기재한다.

8. git push 원격지주소 브런치명 :: 푸쉬함
